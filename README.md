# Local ChatGPT

A simple ChatGPT clone built with Next.js and Tailwind CSS.

##  Features
- Sidebar for multiple chats
- Message streaming
- Retry and Stop options
- Rename, Delete chat
- Fully responsive

##  Tech Stack
- React + Next.js
- Node.js + Express
- Tailwind CSS

##  Setup Instructions
1. Clone the repo
2. Run `npm install`
3. Run frontend: `npm run dev`
4. Backend: Run Express server on port 5000
